# Hood-Inner-and-Outer-Panel_HYPERMESH-Optistruct-
Objective: To mesh the give Hood challenge with the given Criteria , 
Geometry Clean-Up and Mid-Surface generation,
Procedure as follows: Geometry Clean up with Quick Edit / Surface edit
Midsurface Generation : Auto extraction with closed
Recheck with Damaged surface and clean up with Quick edit
2D meshing
Edit the surface with Automesh option
If Number of Tria are more , reduce by splitting and joining with neighbour element
Use rebuild mesh for better mesh flow and Smooth option
For Quick mesh edit tool use QualityIndex option
Element Quality Checks
After Completion of meshing on the Midsurface , Follow the Quality criteria for element quality checks
Use Check elems , for element quality like.. Wrapage, aspect, skew , min&max length jacobian etc..
Check for free edges , if there’s any gap in between the elements using edges option , etc..
